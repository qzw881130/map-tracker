# 运动追踪app 👋

记录运动（遛狗，骑车，跑步，爬山，滑冰，自驾，其他）的起止时间，路程，轨迹和速度

## Get started

1. Install dependencies

   ```bash
   npm install
   npx expo prebuild -p ios
   ```

2. Open ./ios by xcode && iphone connect mac && build 

3. Start serve

   ```bash
    npm start --device
   ```
